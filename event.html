Event

Kesimpulan: kode ini membangun struktur dasar untuk menampilkan koleksi gambar pada halaman "Event" serta menambahkan event Javascript untuk berinteraksi dengan gambar tersebut.
Klik sekali (single click) memunculkan alert "a"
Klik dua kali (double click) memunculkan alert "b"
Klik dan tahan (mousedown) memunculkan alert "c"
Gerakan mouse di atas gambar (mousemove) memunculkan alert "d"
Gerakan mouse keluar gambar (mouseout) memunculkan alert "e"
Gerakan mouse mendekati gambar (mouseover) memunculkan alert "f"
